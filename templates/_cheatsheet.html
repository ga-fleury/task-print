Syntax cheatsheet

Separator

Three semicolons on their own line splits tasks. Each chunk is its own strip with a partial cut.

;;;

Headings

# Big title (2x2, bold)
## Medium heading (2x height, bold)
### Small bold heading

Lists & checkboxes

- bullet
1. numbered
- [ ] task to tick with a pen
- [x] completed task

Bold

**bold text**

Italic and strikethrough are silently dropped on print (ESC/POS limitation).

Blockquote, code, table

> quoted line
```
code block (printed verbatim)
```

Tables work but are tight on 48 columns.

QR codes

```qr
https://example.com
```

Or any payload: WIFI:T:WPA;S:MySSID;P:hunter2;; for Wi-Fi, vCards, plain text.

Images

  • Type ![alt](https://...) for a URL
  • Drag a file into the editor
  • Paste a clipboard image (Ctrl/Cmd + V)

Images are resized to 384px wide before insertion (and again before printing).

Keyboard

Ctrl/Cmd + Enter prints. Esc closes this modal.
